# Phishing-Lab
Este laboratorio interactivo documenta el ciclo de vida completo de un ataque de ingeniería social, desde la fase de engaño inicial hasta la investigación técnica en el sistema afectado. Profundiza en el análisis forense de logs y el aislamiento de amenazas para minimizar el impacto en infraestructuras corporativas.
